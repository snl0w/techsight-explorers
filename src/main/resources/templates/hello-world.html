"Hello World"

Este é o programa mais básico que você pode escrever em Java, e é frequentemente usado como um primeiro exemplo ao aprender a linguagem.

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

Comentários

São trechos de código que são ignorados pelo compilador e não afetam a execução do programa. Eles são usados principalmente para explicar o código, tornando-o mais legível para outros desenvolvedores (ou para você mesmo no futuro). Java suporta três tipos de comentários:

1. Comentários de Linha Única Comentários de linha única começam com //. Tudo que segue após // até o fim da linha é ignorado pelo compilador. Exemplo: // Este é um comentário de linha única

2. Comentários de Múltiplas Linhas Comentários de múltiplas linhas começam com /* e terminam com */. Tudo entre esses delimitadores é ignorado pelo compilador, mesmo que se estenda por várias linhas. Exemplo: /* Este é um comentário de múltiplas linhas. Ele pode ser usado para explicar trechos mais complexos de código ou para adicionar informações detalhadas. */

3. Comentários de Documentação (Javadoc) Os comentários de documentação começam com /** e terminam com */. Eles são usados para gerar automaticamente a documentação do código usando a ferramenta Javadoc. Dentro desse tipo de comentário, você pode usar tags especiais para organizar a documentação.
