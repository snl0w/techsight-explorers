Estrutura de código

1. Declaração de pacote

A declaração de pacote em Java é opcional, mas geralmente recomendada.
Deve ser definida antes de qualquer declaração de classe ou interface.

package nome_pacote;

2. Importação de pacote

A instrução import permite que você acesse diretamente classes e interfaces de outros pacotes, sem a necessidade de especificar seu caminho completo em cada uso. Ela deve ser definida logo após a declaração do pacote (se houver) e antes de qualquer declaração de classe ou interface.

import java.util.Scanner;

3. Definição de classe

A declaração de classe em Java é obrigatória e serve para definir um novo tipo de objeto. Ela começa com o modificador de acesso (como public ou private), seguido pela palavra-chave class, o nome da classe e, opcionalmente, extensões e implementações de outras classes ou interfaces. A declaração de classe deve incluir um corpo entre chaves {}, onde serão definidos os atributos, métodos e construtores que compõem o comportamento da classe.

public class TechSightExplorers { // Método principal public static void main(String[] args) { // Código aqui } }
